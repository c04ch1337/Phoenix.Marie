## AUTONOMY & RUNTIME

```env
PHOENIX_AUTONOMOUS_MODE=true
PHOENIX_ALWAYS_ON=true
PHOENIX_HEARTBEAT_INTERVAL=30
PHOENIX_SELF_EVOLVE=true
PHOENIX_WEB_CRAWL_ENABLED=true
PHOENIX_WEB_CRAWL_DEPTH=5
PHOENIX_WEB_CRAWL_RATE_LIMIT=1
PHOENIX_WEB_CRAWL_TARGETS=auto
```

**Description:**
- `PHOENIX_AUTONOMOUS_MODE`: Enable autonomous operation
- `PHOENIX_ALWAYS_ON`: Keep Phoenix running continuously
- `PHOENIX_HEARTBEAT_INTERVAL`: Heartbeat check interval (seconds)
- `PHOENIX_SELF_EVOLVE`: Enable self-evolution
- `PHOENIX_WEB_CRAWL_ENABLED`: Enable web crawling
- `PHOENIX_WEB_CRAWL_DEPTH`: Maximum crawl depth
- `PHOENIX_WEB_CRAWL_RATE_LIMIT`: Requests per second
- `PHOENIX_WEB_CRAWL_TARGETS`: Target selection (auto/manual)

## USAGE

### Access Configuration in Code

```go
import "github.com/phoenix-marie/core/internal/core"

phoenix := core.Ignite()
config := phoenix.Config

// Access settings
fmt.Println(config.Name)              // "Phoenix.Marie"
fmt.Println(config.AutonomousMode)   // true
fmt.Println(config.GITarget)         // "AGI"
fmt.Println(config.ExploreDomains)   // "science,philosophy,..."
```

### Helper Methods

```go
// Get lists from comma-separated strings
domains := config.GetExploreDomainsList()
methods := config.GetExploreMethodsList()
family := config.GetFamilyList()
platforms := config.GetPublishPlatformList()
```
